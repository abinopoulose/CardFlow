# CardFlow

CardFlow is a fast, browser-based web application for designing and generating bulk ID cards, badges, and passes. It features a drag-and-drop visual editor and allows you to import spreadsheet data via CSV or Excel to automatically map and generate hundreds of unique cards at once.

**Access the application:** [cardflow.abino.in](https://cardflow.abino.in)

**Privacy Focused:** CardFlow operates entirely on the client side. All processing, data mapping, image rendering, and PDF generation happen directly within your browser. No spreadsheets, photos, or personal data are ever uploaded or sent to any external server.
